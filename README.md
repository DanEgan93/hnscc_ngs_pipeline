<h1> CWL-Docker NGS analysis pipeline for SE Illumina data </h1>

<p>
	A CWL co-ordinated pipeline to run NGS tools within docker containers. The tools below are included in the NGS pipeline.

	1. Fastqc
	2. Trimmomatic
	3. Bowtie2
	4. Samtools
	5. Vardict
	6. VCFtools
	7. Bedtools
	8. VEP
</p>
